Repositorio criado para testar o que foi aprendido no curso da code.education do modulo git

PASSOS EXECUTADOS

1 - Criei uma pasta com o arquivo index.php
2 - Executei o comando git init para iniciar o versionamento do projeto
3 - Criei o arquivo README.md e o arquivo index.php
5 - Executei o comando git add . para adicionar todos arquivos que serão 
controlados pelo git
6 - Executei o comamando git commit -m "" para comitar todos arquivos da pasta
7 - Criei no github um repositorio para associar ao meu repositorio local
